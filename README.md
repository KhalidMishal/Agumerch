# Agumerch
AGU (Abdullah Gul University) Store Application project

A collaborative mobile programming project aimed at promoting and supporting AGU's official merchandise.
It will be devleoped using Flutter and Dart to run on both Android and Web platforms. 
The application allows students, staff, and alumni to explore and purchase official 
AGU-branded clothing and accessories through a modern, intuitive interface

By the end of the project, the team will deliver a fully functional prototype of a digital
merchandise store for AGU University. The application will allow users to create accounts,
browse and view university-branded products in interactive 3D, add items to a cart, and
complete a simulated purchase process.
